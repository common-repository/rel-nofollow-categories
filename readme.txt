== Description == 

Rel Nofollow Categories is a plugin which adds nofollow attribute to categories generated by the_category and wp_list_categories throughout the blog.

Adds Nofollows to category links and an option to noindex the category pages.

Plugin by <a href="http://tutskid.com/">How To | Tutorials | TutsKid</a>.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Activate the plugin through the 'Plugins' menu in WordPress.

OR

Just grab Rel Nofollow Categories from the built in plugin directory and choose "Install now", hit ok, then select the "Activate Plugin" link.
